# clojure-noob

Examples and exercise following Clojure for the Brave and True

## Usage

In REPL:

```
(use 'clojure-noob.core :reload)
(-main)
attack hobbit!
{:name right-ear, :size 1}
attack hobbit!
{:name chest, :size 10}
attack beast!
{:name second-ear, :size 1}
attack crazy beast!
{:name thigh-92, :size 4}
=> "Yahhh!"
```

Hit some hobbits and beasties and other fun!
